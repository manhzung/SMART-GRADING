\begin{center}
    \Large{\textbf{ABSTRACT}}\\
\end{center}
\vspace{1cm}
In the context of large-scale multiple-choice examinations becoming increasingly common in secondary schools and universities, manual grading of paper-based answer sheets still suffers from many limitations including time consumption, proneness to errors, lack of transparency, and difficulty in statistical analysis. Existing automated grading solutions often rely on expensive specialized scanning hardware or only address a partial workflow such as question generation or answer sheet recognition, without providing an integrated system covering the entire process from question bank management and exam versioning to OMR-based grading on mobile devices, transparent appeal handling, and comprehensive reporting.

This thesis adopts the approach of building a multi-platform smart multiple-choice grading system that integrates Optical Mark Recognition (OMR) on mobile devices, automatically generates multiple exam versions from a shared question bank, supports a transparent appeal workflow, and provides detailed statistical reports for teachers and administrators. The system follows a three-tier architecture comprising a Node.js/Express backend with MongoDB database, a Flutter mobile application that leverages the device camera for scanning and grading, a React web dashboard for administration and analytics, combined with the AMC (Auto-Multiple-Choice) LaTeX package for automatic exam paper and answer sheet generation. The system also integrates large language models from Gemini, OpenAI, and Claude to assist teachers in building the question bank.

The main contribution of the thesis is the complete Xây dựng hệ thống quản lý ngân hàng câu hỏi và chấm thi trắc nghiệm tự động sử dụng công nghệ OMR system with capabilities for managing schools, classes, teachers, and students; a question bank and exam management module with AI-assisted question generation; automatic generation of multiple exam versions and corresponding OMR answer sheets through AMC with precise recognition coordinates; mobile OMR-based automatic grading with high accuracy, including detection of double marks and light marks, with manual override capability; a transparent appeal submission and response workflow with real-time notifications; as well as detailed per-exam reports and analytics. The system has been deployed in two classes with around 60 students over four weeks, successfully processing 5 exams and approximately 300 submissions, achieving an average OMR accuracy of 96.2\% and an average API response time of 180ms.
